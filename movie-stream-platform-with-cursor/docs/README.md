# ドキュメント一覧

本ディレクトリには、動画配信プラットフォームの開発に関するドキュメントを格納しています。

## ドキュメント構成

| ファイル | 説明 |
|----------|------|
| [requirements.md](./requirements.md) | 要件定義書。アプリの機能・画面・データ構造を定義 |
| [progress.md](./progress.md) | 進捗管理。タスクの状態を追跡し、開発進捗を管理 |
| [tech-stack.md](./tech-stack.md) | 技術スタック。使用する技術・ライブラリの一覧 |

## 開発の流れ

2. **tech-stack.md** で技術選定を把握
3. **progress.md** でタスクを進め、完了したら ✅ に更新
